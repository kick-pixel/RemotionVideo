# 深入解密 React Server Components (RSC)

React Server Components 是一项革命性的技术。传统的 React 组件（CSR）需要将所有 JS 代码下发到了客户端，导致首屏加载慢，并且打包体积越来越大。

RSC 的核心优势：
1. **零 JS 下发**：服务端组件在服务器渲染完成，客户端只接收类似 HTML 的精简标记，不包含任何 JS，大大减少打包体积。
2. **直接访问数据库**：由于服务端组件运行在服务端，你可以直接在组件里写 `await db.query()` 获取数据，不需要再写额外的 API 层！
3. **快速响应**：大大提升了页面的加载性能和响应速度。
